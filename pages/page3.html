Google опублікувала Android 9 Pie
6.08.2018 23:13

6 серпня 2018 року корпорація Google оголосила про реліз нової версії операційної системи Android. У тестуванні розробки взяли участь понад 140 тисяч учасників програми Android Beta. Нововведення торкнулися і простих користувачів, і розробників. Вихідні коди Android Pie було викладено в Android Open Source Project (AOSP).

Що нового в Android 9 Pie?

Машинне навчання понад усе

Android 9 автоматично налаштовує телефон залежно від уподобань і звичок користувача для максимального часу роботи та кращої продуктивності найчастіше використовуваних додатків.

Adaptive Battery

Google уклала партнерство з DeepMind для роботи над функцією під назвою Adaptive Battery, яка використовує машинне навчання для визначення пріоритетів системних ресурсів.

App Actions

App Actions - нові методи підвищення наочності застосунку. Вони використовують переваги машинного навчання, щоб спрямувати необхідний застосунок користувачеві в потрібний час, ґрунтуючись на семантичних намірах застосунку та контексті користувача.

Класифікатор тексту та лінковщик Smart Linkify

Розробники розширили моделі машинного навчання, які ідентифікують об'єкти у вмісті або введенні тексту, щоб підтримувати такі типи, як дати та номери рейсів, через API TextClassifier. Smart Linkify дає змогу використовувати моделі TextClassifier через API Linkify, включно з розширеними опціями для швидкої подальшої дії користувача. Smart Linkify забезпечує поліпшення точності виявлення, а також продуктивності.

API Neural Networks 1.

Android 9 привнесла оновлену версію API для нейронних мереж, щоб розширити підтримку Android для прискореного навчання пристрою на пристрої. Neural Networks 1.1 додає підтримку для дев'яти нових операційних систем - Pad, BatchToSpaceND, SpaceToBatchND, Transpose, Strided Slice, Mean, Div, Sub і Squeeze. Типовим способом застосування API є використання TensorFlow Lite.

Нова система навігації

Розроблювана понад рік система спрощує взаємодію користувача з пристроєм, забезпечуючи швидкий та інтуїтивно зрозумілий перехід між екранами.

Text Magnifier

Віджет Magnifier дає змогу користувачам точно позиціонувати курсор або маркери вибору тексту, переглядаючи збільшений текст через перетягувану панель. Він також може надавати збільшену версію будь-якого виду або поверхні, а не тільки текст.

Біометрична аутентифікація

Додатки, які раніше вже використовували підтвердження особи користувача за допомогою біометричних даних, тепер можуть використовувати вбудований BiometricPrompt API.
