Вийшло оновлення мови Dart 2.0, альтернативи JavaScript від Google
08.08.2018 10:58

Розробники з Google опублікували версію мови програмування Dart 2.0 для мобільних і веб-систем. У новому випуску з'явилася сувора статична типізація, але типи можуть виводитися автоматично та їхні анотації як і раніше, не є обов'язковими.

Компіляція

У Dart 2 режим "strong" для суворої перевірки типів замінив режим "checked" для їхньої обмеженої перевірки. Сувора типізація і компілятор dartdevc забезпечують швидку компіляцію в JavaScript без віртуальних машин. Для тестування додатків тепер можна використовувати компіляцію в JavaScript і будь-які штатні браузери замість Dartium.

Робота з інтерфейсом

У новій версії мови можна визначати віджети для інтерфейсу без обов'язкового використання ключових слів new і const. Раніше для цього потрібно було застосовувати окрему мову розмітки.

Розробка додатків

Для створення мобільних додатків можна використовувати фреймворк Flutter. Для веб-розробки творці Dart 2 пропонують набір специфічних бібліотек, таких як dart:html і Angular. Також є можливість створювати однорідні системи з єдиною загальною кодовою базою.

Зростання популярності Dart

У рейтингу TIOBE за серпень 2018 року мова програмування посідає 24 місце (докладніше про серпневий рейтингу можна прочитати в нашому матеріалі). Також у рейтингу GitHub за другий квартал 2018 року Dart посідає 23 позицію.

У лютому 2018 року було представлено бета-версію Dart 2.0. Крім того, Google активно працює над операційною системою Fuchsia, написаною на Dart. У січні 2018 року Google опублікувала на GitHub інструкцію зі встановлення Fuchsia OS, а вже у квітні стало можливим випробувати демоверсію системи прямо в браузері. А в липні редакції Bloomberg стало відомо, що Google планує впровадити Fuchsia OS в IoT-пристрої до 2021 року.
